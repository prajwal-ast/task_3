📌 Overview

This project demonstrates a vulnerability assessment performed on a Kali Linux host using Nessus Essentials. The objective was to identify known vulnerabilities, evaluate the security posture of the system, analyze the severity of the findings, and recommend appropriate remediation measures.


🎯 Objectives
Install and configure Nessus Essentials.
Perform a vulnerability assessment on the local host.
Identify and categorize vulnerabilities based on severity.
Analyze the security impact of detected vulnerabilities.
Recommend remediation strategies to reduce security risks.
Document the assessment in a professional report.
